# OsrsCalcTool.Api

A .NET 10 Web API providing OSRS Grand Exchange pricing, hiscores, and item data.

## Development

```bash
dotnet run --project OsrsCalcTool.Api
```

API will be available at `https://localhost:7xxx`. Scalar API reference is available at `/scalar` in development.

## Building

```bash
dotnet build
dotnet publish -c Release
```
